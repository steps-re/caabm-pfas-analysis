## Abstract

Ni et al. (2026) report a coagulant-assisted algae-bubble matrix (CAABM) that removes PFAS using oxygen bubbles generated in situ by microalgal photosynthesis, achieving 7–14% removal in 97 min and a reported rate enhancement up to 428× over prior algal methods. Reconstructing the governing physics from first principles and reconciling it against the authors' own supplementary data yields three conclusions:

1. **The process is flotation, not adsorption.** The air-water interface is the sorbent; the algae manufacture and stabilize it. The authors' batch tests (no removal without bubbles) establish this directly.
2. **Foam fractionation is physically barred from short-chain PFAS, and the coagulant works by switching mechanisms.** Air-water interfacial adsorption falls ~3× per carbon, putting PFBA ~50× below PFOA. If foam fractionation were the mechanism, PFOA would out-remove PFBA ~50×; the authors observe ~1.2×. The coagulant converts a chain-length-dependent *interfacial* process into a nearly chain-length-independent *electrostatic* one, which is why short-chain gets the 428× and long-chain only 21×.
3. **Cell surface charge is the decisive selection variable.** Chlorella (−24.5 mV) removed 0%; Ankistrodesmus (−13.8 mV) removed 4–14%. The 100 ppm pDADMAC over-dose that fell to ~1.5% is the same charge law running in reverse.

The authors' proposed multi-decade search for a better organism is recast as a **deterministic screen** over four measurable traits, with a reproducible scoring function and candidate shortlist. The screen ranks *Ankistrodesmus* above *Chlorella* sp., retrodicting the observed CAABM result.

## Reproduce every number

No third-party dependencies (Python 3.8+ standard library only):

```bash
cd code
python interfacial_adsorption.py   # chain-length ladder; Γ_PFOA/Γ_PFBA ≈ 36–90×
python flotation_ceiling.py        # PFOA ceiling brackets 4.4%; PFBA ceiling << observed
python dlvo_barrier.py             # Chlorella barrier 1.8–3.2× higher than Ankistrodesmus
python species_screen.py           # ranks 22 candidate strains; retrodicts Ankistrodesmus > Chlorella
python surface_predictor.py        # tests genome/cell-wall -> hydrophobicity (returns a clean negative result)
python staging_model.py            # how ~10-14%/pass compounds to >90% removal over N stages
python pfas_qsar.py                 # classifies all 16 PFAS foam-removable vs charge-dependent (logKow r=0.99)
```

## Data provenance & honesty

- Experimental values are **extracted from the machine-readable SI of Ni et al. (2026)** and cited per-row in the CSVs. Raw supplementary files are **not redistributed**; only the numeric values used in this analysis are reported, as is standard for a citing work.
- Physical constants and literature parameters carry sources in `constants.py` and `references.bib`.
- Every DOI in `references.bib` was verified against Crossref. Six citation errors in an earlier draft were caught and corrected (noted inline in the .bib).
- Order-of-magnitude estimates (the absolute removal ceiling) are labeled as such; the mechanism conclusion rests on a **ratio** that is independent of the uncertain inputs.
- Two screen features (`bubble_score`, `lowEPS_score`) are coarse literature-derived encodings, not measured values, and are flagged in `data/README.md`.
